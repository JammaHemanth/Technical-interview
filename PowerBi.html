1. What is Power Bi?
It was first released to the general public on 24 July 2015 it has several versions for desktop,mobile,web app
Micosoft power bi is a data visualization platform used primarily for business intelligence purpose
Power Bi dashboard is capable of reporting and visualizing data in a wide range of different styles,including graphs,maps,charts,scatter plots and more

2. What is the use of Power Bi?
It provides a simple set up to connect transform and visualize the data and use it to make better decision

3. What is Power Query?
It is an ETL tool that extracts,loads and transform data
It can retrive data from sources,process it and load it into one or more target systems

4. What is Power pivot?
It is a data modelling technique that lets you to create a data models,establish relationship and create calculations
It was data analysis expression(DAX) language to model simple and complex data

5. What is Power View?
It is a data visualization tool that allows user to create interactive charts,graphs,maps and other visuals
It is available in excel,sharepoint,SQL server and Power BI

6. What is Power Bi Service?
It is a cloud-based service, or software as a service(SaaS)
It suppors report editing and collabration for team and organizations

7. What is Power Bi desktop?
It is a free application that allows user to create dashboard and live reports it can be installed on a computer to connect to transform and visualize data

8. What is Power Bi dashboard?
It is a canvas that helps in creating a story with visualization and templates to better understand the data

9. What is Dax Function that is used in Power BI?
Dax Function is a formula expression language used with various business intelligence visualization tools like Power BI. It is also known as a functional language, where the full code is kept inside a function.

10.What is Visualization in Power BI?
Visualization is a process of Power BI used to represent data in pictorial form like tables, graphs or charts based on the specific requirement.

11.What is a Report in Power BI?
The Report is a feature of Power BI, resulting from visualized data from a single data set. A report can contain multiple pages of visualization.

12.What are KPIs in Power BI?
KPIs stands for Key Performance Indicators. KPIs are used to evaluate the organization's performance in distinct areas by evaluating measurable goals and values.
A KPI contains a measure or base value evaluated against target values and compares the performance with the target.

13.What is the data source filter in Power BI?
A data source filter is used to filter the data before loading it into machines. It is a parameter used for data filtering.
