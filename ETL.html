4. What is 3-Layer architecture?
i)Staging Layer: It is used to store the data which is extracted from the different data source system.
ii)Data Integration: It transforms data from the staging layer and move the data to the database. The combination of fact and dimension table in a data warehouse system is called schema. In this database, tha data is arranged into hierarchical group which is called dimension.
iii)Access layer: It is used to the end users to retrieve data for analytical reporting.
