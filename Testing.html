1. What is Software Testing?
It is a Validation Process that confirms that system works as per the business requirement

2. What is a test case?
It is a document that has set of conditions or actions in order to verify the expected functionality of the feature

3. What is test scenario?
It gives an idea of what we have to test and it is like high level test case

4. What is test plan?
It stores all possible testing activites to ensure a quality product
It gather data from the product description,requirements and use case documents

5. What is Functional testing?
It is a testing technique which checks the functionality of an application works under the required specification

6. What is Non-Functional Testing?
It checks all the non-functional aspects such as perfomance,usability and reliability etc

7. What is Sanity Testing?

33. What is the difference between Load and Stress Testing?
I) In Load testing we are testing the software at its maximum limit. The basic purpose of Load testing to find out how many maximum user applications can handle without losing the performance.
II) In Stress Testing we are testing the software beyond its limit to check the behavior of application when it crosses the maximum limit of user.

36.What is Application Programming Interface (API)?
